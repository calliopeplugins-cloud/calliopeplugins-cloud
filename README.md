# CalliopeWP

CalliopeWP is the WordPress plugin line developed by Estudio Calliope from Rosario, Santa Fe, Argentina.

We build practical tools for WordPress and WooCommerce projects, with a focus on SEO, content workflows, automation, media management, and ecommerce operations.

## Plugin Catalog

- Plugin catalog: https://pluginswordpress.calliope.com.ar/
- Institutional site: https://calliope.com.ar/
- Technical contact: contacto@calliope.com.ar

## Project Status

This GitHub profile is used to organize CalliopeWP plugin development and maintenance. Public repositories may be published when a Free edition is ready for open distribution and has completed the required security and release checks.

Private repositories, internal tools, licensing systems, and Pro editions are not listed here unless they are intentionally made public through official CalliopeWP channels.

## Free and Pro Editions

Some CalliopeWP products may have separate Free and Pro editions. They are maintained as separate codebases and repositories when both editions exist.

Pro products are distributed only through the official CalliopeWP channels and support processes.
