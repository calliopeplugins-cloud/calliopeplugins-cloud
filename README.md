# CalliopeWP

CalliopeWP develops practical WordPress and WooCommerce plugins as part of Estudio Calliope, a web development and digital product studio based in Rosario, Santa Fe, Argentina. The public repositories in this GitHub account contain the Free editions of selected CalliopeWP plugins, with source code, repository documentation, contribution guidelines, and security reporting information for users and developers.

## Why CalliopeWP

CalliopeWP focuses on tools that fit naturally into the WordPress admin. The Free editions are kept visible here so users can inspect the public code, understand the plugin structure, follow changes through pull requests, and review security and contribution policies. The goal is to keep each plugin focused on a real workflow: SEO, media management, product pricing, sales documents, or social publishing.
